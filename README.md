### 🎯 Objective
Classify Iris flowers into 3 species using Machine Learning + Data Visualization

### 🛠️ Tech Stack

### 📊 What I Did
1. 📥 Loaded Iris dataset 
2. 📈 Created 4 graphs: 
   - Countplot for species distribution
   - Heatmap for feature correlation 
   - Pairplot for feature relationships
   - Confusion Matrix for model results
3. 🤖 Built Logistic Regression model
4. ✅ Achieved ~96% accuracy
